Para la instalación de los módulos:
1. matplotlib
2. numpy
3. pandas

En el caso de windows:
1. acceder a cmd (consola de windows)
2. introducir el siguiente comando (si se tiene la versión Python 3.x):
	pip3 install módulo
	donde módulo=numpy, matplotlib, pandas.
Ejemplo:
Para instalar numpy el comando en cmd es:
	pip3 install numpy
Nota: en caso de tener la versión Python 1.x el comando es:
	pip install numpy
